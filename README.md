- CSS
- Redux

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

Basic JavaScript
Reactjs
Eslint
### Setup

clone the project
run npm install
run npm start

### Usage

Start server with:

```
npm start

```

Unit Test:

```
npm run test

```
